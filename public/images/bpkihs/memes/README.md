Add approved meme images for the BPKIHS wrong-answer screen in this folder.

Recommended formats:
- `.webp` for smallest files
- `.jpg` for photos/screenshots
- `.png` only when transparency or crisp text is needed

After adding an image, register it in:

`src/data/bpkihsMemes.js`

Example:

```js
{
  id: "viva-table-judging",
  src: "/images/bpkihs/memes/viva-table-judging.webp",
  alt: "Funny viva table judging meme"
}
```

Avoid hotlinking social-media images. Keep the image files inside this repo so the page stays fast and reliable.
